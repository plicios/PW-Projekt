To build project, specify DllName and DatabasePath in AppConfig of KetchupCatalogUI. There is an example app config in file App.config.example.

DllNames:
DaoDatabase (with good connection string and database it get and write values from and to database)
DAOMock 	(Mock of database, data not stored between program run)
DAOMock2 	(Mock of database number 2, data not stored between program run)
